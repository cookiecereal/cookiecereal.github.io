Employee SQL Dashboard Project

Project Overview

In November 2023, I developed a comprehensive SQL dashboard integrated with Excel to analyze employee data, answer complex scenario questions, and extract valuable insights. This personal project, based in El Paso, TX, showcases my skills in SQL, data analysis, and dashboard creation.

Key Features

  • SQL-Based Analysis: Developed a robust SQL dashboard to process and analyze employee data.
  • Excel Integration: Seamlessly integrated SQL query results with Excel for enhanced visualization and reporting.

  • Data Management: Enabled the addition and deletion of entries to maintain data accuracy and relevance.

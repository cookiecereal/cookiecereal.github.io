Grad Grant System Project

Project Overview

The Grad Grant System is a comprehensive platform designed to connect researchers with appropriate funding opportunities. This system streamlines the process of finding, applying for, and managing research grants, catering to individual researchers, research teams, and institutions.

Key Features

  • Personalized Funding Recommendations: Displays funding opportunities based on registered users' interests.
  • User Management: Supports researchers with detailed profiles including name, contact information, email, and area of expertise.
  • Solicitation Management: Manages funding opportunities with detailed information including name, CFTA number, eligibility, and requirement documents.
  • Application Process: Facilitates the application process for multiple grants, including checklist generation.
  • Admin Controls: Allows administrators to change a researcher's award status.

Technical Details

User Management

  • Implemented user profiles with roles (e.g., researcher, team member, PI) and detailed information.
  • Developed logic to enforce restrictions such as PI limitations and junior faculty-specific awards.
  • Created a system to manage team compositions and apply team-specific restrictions.

Solicitation and Proposal Management

  • Designed a database to store and manage solicitation details, including eligibility criteria and funding limits.
  • Implemented a proposal submission system with deadline enforcement.
  • Developed an automated checklist generation system for applicants.

Key Achievements

  • Efficient Grant Discovery: Streamlined the process of finding relevant funding opportunities for researchers.
  • Customized User Experience: Provided personalized recommendations based on user profiles and interests.
  • Streamlined Application Process: Simplified the grant application process with guided checklists and deadline management.
  • Flexible Administration: Enabled administrators to manage user statuses and oversee the grant allocation process.

Technologies Used

  • Database: MySQL, PostgreSQL
  • Backend: Java, Python, Node.js
  • Frontend: React, Angular, Vue.js

Challenges and Solutions

  • Challenge: Managing complex eligibility criteria for different grants.
    Solution: Implemented a flexible rule-based system for defining and checking eligibility.
  • Challenge: Ensuring data privacy and security for sensitive research information.
    Solution: Implemented robust encryption and access control measures.
  • Challenge: Handling multiple simultaneous applications and team compositions.
    Solution: Developed a scalable database structure and efficient querying system.

Future Enhancements

Potential areas for future development include:

  • Implementing an AI-driven recommendation system for more accurate funding matches.
  • Developing an integrated collaboration platform for research teams.
  • Creating a mobile application for on-the-go access to grant information and applications.

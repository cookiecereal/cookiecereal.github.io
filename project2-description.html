Grad Grant System Project

Project Overview

Developed at the University of Texas at El Paso, the Grad Grant System is a comprehensive solution designed to streamline the process of connecting researchers with appropriate funding opportunities. This project showcases my ability to create data-driven systems that enhance academic processes.

Key Features

  • Personalized Funding Recommendations: Created a system to display funding opportunities for researchers based on their interests and qualifications.
  • Advanced Data Processing: Transformed and filtered data using aggregating and filtering functions to improve the reporting process.
  • Administrative Dashboard: Loaded and visualized data with a frontend interface for administrators to view reports on researchers who have been granted funding.

Technical Details

Data Processing and Analysis

  • Implemented advanced data transformation techniques to clean and structure raw funding data.
  • Utilized aggregation functions to summarize large datasets into meaningful insights.
  • Developed custom filtering algorithms to match researchers with relevant funding opportunities.

Frontend Development

  • Designed an intuitive user interface for researchers to explore funding opportunities.
  • Created an administrative dashboard for visualizing grant allocation data.
  • Implemented responsive design principles to ensure accessibility across various devices.

Outcomes and Impact

  • Streamlined the grant discovery process for UTEP researchers.
  • Improved the efficiency of the university's research funding allocation.
  • Provided administrators with valuable insights into funding trends and researcher interests.
